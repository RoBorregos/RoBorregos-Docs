## Add new page

To add a new page, locate the docs directory.
```{bash}
ROBORREGOS-DOCS
│   mkdocs.yml
│   requirements.txt
│
└───docs
    │   index.md
    │   YourNewPage.md
    └───assets
            favicon.png
            logo.png
```
Here, add a new **.md** file and add your content.
